# Medical Misinformation Detection using NLP

## Project Overview

This project performs a comparative analysis of NLP models to detect healthcare misinformation.

The study evaluates:

* Logistic Regression (TF-IDF based)
* BERT (HuggingFace pretrained model)

## Dataset

Medical Misinformation Dataset:
https://github.com/kinit-sk/medical-misinformation-dataset

## Models Used

1. Logistic Regression
2. BERT (DistilBERT)

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1-score
* Confusion Matrix

## Tools & Technologies

* Python
* Google Colab
* Scikit-learn
* HuggingFace Transformers
* Pandas
* Matplotlib

## How to Run

1. Open notebook in Google Colab
2. Run cells sequentially
3. Results will be generated automatically

## Output

The project compares traditional machine learning with transformer-based deep learning models for healthcare misinformation detection.
